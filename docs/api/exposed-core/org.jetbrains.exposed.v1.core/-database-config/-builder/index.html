var defaultMaxAttempts: Int

The maximum amount of attempts that will be made to perform any transaction block. If this value is set to 1 and a database exception happens, the exception will be thrown without performing a retry. This can be overridden on a per-transaction level by specifying the maxAttempts property in a transaction block. Default amount of attempts is 3.

LocalDateColumnType

abstract class LocalDateColumnType<T> : ColumnType<T> , IDateColumnType

Base column type for storing date values without time components.

This abstract class provides the foundation for date-only columns across different database dialects. It handles the conversion between application date types and database representations, ensuring proper formatting and parsing based on the current database dialect.

Parameters

T

The application-specific date type (e.g., LocalDate, kotlinx.datetime.LocalDate)

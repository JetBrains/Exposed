IntEntityClass

abstract class IntEntityClass<out E : IntEntity>(table: IdTable<Int>, entityType: Class<E>? = null, entityCtor: (EntityID<Int>) -> E? = null) : EntityClass<Int, E>

Base class representing the EntityClass that manages IntEntity instances and maintains their relation to the provided table.

Parameters

table

The IdTable object that stores rows mapped to entities of this class.

entityType

The expected IntEntity type. This can be left null if it is the class of type argument E provided to this IntEntityClass instance. If this IntEntityClass is defined as a companion object of a custom IntEntity class, the parameter will be set to this immediately enclosing class by default.

entityCtor

The function invoked to instantiate an IntEntity using a provided EntityID value. If a reference to a specific constructor or a custom function is not passed as an argument, reflection will be used to determine the primary constructor of the associated entity class on first access. If this IntEntityClass is defined as a companion object of a custom IntEntity class, the constructor will be set to that of the immediately enclosing class by default.
